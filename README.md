# C-Programming-project-

This is my First C programming code after learning 2 weeks.

This project is a simple C program that gives a virtual tour of a house using a menu system.

The user can choose different rooms to visit such as the kitchen, living room, guest room, or upstairs.

- Menu based navigation
- Different house rooms
- User input using keyboard
- Exit option

1. Kitchen
2. Living Room
3. Upstairs
4. Guest Room
5. Exit


## How to Run
Compile the program:

gcc house_tour.c -o house

Run the program:

./house
